# Firefox Add-ons (AMO) Listing

## Summary (max 250 chars)
Restores the inline shoutbox on Last.fm artist, album, track, and user pages. Read shouts, post new shouts, and reply to existing ones — all without leaving the page.

## Description
<p>Last.fm used to display a shoutbox inline on every artist, album, track, and user profile page. That UI is gone from the site, but the underlying shout data still exists. This extension brings the shoutbox back — rendered directly on the page, right where it used to be.</p>

<p><strong>What it does:</strong></p>
<ul>
  <li>Displays shouts inline on Last.fm artist, album, track, and user pages</li>
  <li>Lets you post new shouts</li>
  <li>Lets you reply to existing shouts</li>
  <li>Works without leaving the page or opening any popup</li>
</ul>

<p>This extension reads Last.fm's own shout data by parsing the pages Last.fm already serves. It does not use a separate backend, does not store your data, and does not send anything to any third-party server.</p>

## Privacy Policy
This extension does not collect, store, or transmit any user data. It only communicates with last.fm — the site it runs on — to read and post shouts on your behalf when you interact with the shoutbox UI. No analytics, no tracking, no third-party servers.

Source code is publicly available at https://github.com/faustseele/return-lastfm-shoutbox.
