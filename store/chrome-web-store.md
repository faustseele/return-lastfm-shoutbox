# Chrome Web Store Listing

## Short Description (max 132 chars)
Restores the inline shoutbox on Last.fm artist, album, track, and user pages. Read shouts, post shouts, and reply.

## Detailed Description
Last.fm used to display a shoutbox inline on every artist, album, track, and user profile page. That UI is gone from the site, but the underlying shout data still exists. This extension brings the shoutbox back — rendered directly on the page, right where it used to be.

What it does:
- Displays shouts inline on Last.fm artist, album, track, and user pages
- Lets you post new shouts
- Lets you reply to existing shouts
- Works without leaving the page

This extension reads Last.fm's own shout data by parsing the pages Last.fm already serves. It does not use a separate backend, does not store your data, and does not send anything to any third-party server.

It is free and open source under the MIT license.
Source code: https://github.com/faustseele/return-lastfm-shoutbox
